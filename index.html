Notas de accesibilidad

  • El buscador usa role="search" y <label> asociado.
  • El campo de búsqueda sugiere solo títulos existentes con <datalist>.
  • Los <iframe> tienen title descriptivo.
  • Las imágenes tienen alt para lectores de pantalla.
  • La navegación usa anclas (#watch-1, etc.) para simular “cambio de página” sin JS.

Acerca de

Prototipo tipo “Clon de YouTube” realizado con HTML5 semántico exclusivamente. Restricciones: sin CSS, sin JavaScript. La tabla se usa solo para aproximar un layout similar.

Elementos vacíos usados: <meta>, <img />, <hr />.

Búsqueda: se implementa con sugerencias nativas (<datalist>) y solo lista videos existentes.
